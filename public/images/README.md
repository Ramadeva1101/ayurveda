# Ayurvedic Wellness House — Image Assets

All placeholder images in this directory must be replaced with real client-provided photography before launch. Placeholder aspect ratios and labels are preserved in the HTML so layout will not shift when photos are dropped in.

## Image Requirements

### Format
- WebP for all photography (better compression + quality)
- PNG only for logos and transparent graphics
- Max 200KB per photo; optimize before upload

### Photography Style
- Warm, natural lighting — avoid harsh flash
- Earth tones, green plants, wooden textures
- Professional but approachable; human warmth in frame
- No generic stock-photo aesthetic
